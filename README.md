# petsitter-app

# Que es petsitter?
Es una plataforma digital que conecta dueños de mascotas con cuidadores confiables, ofreciendo servicios como paseos, alimentacion y acompañamiento

# Que problema intenta resolver?

Busca solucionar la dificultad que tienen muchos dueños de mascotas cuando necesitan ausentarse por trabajo, viajes o compromisos y no cuentan con alguien de confianza para cuidar a sus animales. Tambien responde a la necesidad de encontrar cuidadores verificados que garanticen seguridad y bienestar

# Quienes son los usuaios principales?
 - Dueños de mascotas que requieren apoyo para el cuidado diario o por momentos

 - Cuidadores o paseadores que ofrecen sus servicios y buscan una forma segura de conectarse con clientes

# Funcionalidades 
Landing principal con información del servicio
Inicio de sesión para cuidadores
Registro de cuidadores con selección de tipo de mascota y experiencia

# Tecnologias utilizadas
- HTML
- CSS
- JAVASCRIPT
